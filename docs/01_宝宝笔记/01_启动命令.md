# FractalFrame_v5 启动命令

## 1. 启动 MT5 本地后端桥

在项目根目录执行：

```powershell
python scripts\mt5_symbols_server.py --host 127.0.0.1 --port 8765
```

这个窗口保持打开。它负责连接本机 MT5，并提供右侧数据中心 `Scan MT5` 需要的品种列表接口。

## 2. 启动前端页面

另开一个终端执行：

```powershell
npm run dev -- --host 127.0.0.1 --port 5185
```

然后打开：

```txt
http://127.0.0.1:5185/
```

使用前先确认 MT5 终端已经打开并登录。
