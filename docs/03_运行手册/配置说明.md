# 配置说明

## 后端 bridge

启动命令：

```powershell
.\.venv\Scripts\python.exe scripts\mt5_symbols_server.py --host 127.0.0.1 --port 8765
```

参数：

- `--host`：监听地址，默认 `127.0.0.1`。
- `--port`：监听端口，默认 `8765`。
- `--cache-root`：MT5 symbol 缓存目录。
- `--store-root`：StoreV5 数据根目录；不填时使用项目默认 runtime 目录。

环境变量：

- `FRACTALFRAME_CORS_ORIGIN`：CORS origin，默认 `*`。
- `FRACTALFRAME_LOG_LEVEL`：日志级别，默认 `INFO`。
- `FRACTALFRAME_JOB_SNAPSHOT_ROOT`：job snapshot 持久化目录，默认 `runtime_data/jobs`。
- `FRACTALFRAME_LOG_ROOT`：日志目录，默认 `runtime_data/logs`。

## 前端

参考 `frontend/.env.example`：

```env
VITE_FRACTALFRAME_MARKET_DATA_HTTP_BASE=http://127.0.0.1:8765
```

## npm warning

如果出现：

```text
npm warn Unknown env config "min-release-age"
```

通常来自父环境变量 `NPM_CONFIG_MIN_RELEASE_AGE`，不是项目配置。`scripts/check_all.ps1` 会在自身进程内清理该变量。
