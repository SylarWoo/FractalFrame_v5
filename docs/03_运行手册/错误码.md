# 错误码

## 通用

- `bad_request`：请求参数缺失或非法。
- `not_found`：路由不存在。
- `job_not_found`：任务 ID 不存在或已被清理。

## MT5

- `mt5_initialize_failed`：MT5 初始化失败。
- `mt5_symbol_select_failed`：品种选择失败。
- `mt5_m1_check_unavailable`：Python 环境缺少 MetaTrader5 包。
- `mt5_m1_check_init_failed`：M1 检查初始化失败。
- `mt5_m1_check_symbol_select_failed`：M1 检查选择品种失败。
- `mt5_m1_check_exception`：M1 检查过程异常。
- `mt5_realtime_unavailable`：实时行情依赖不可用。
- `mt5_realtime_init_failed`：实时行情初始化失败。
- `mt5_realtime_symbol_select_failed`：实时行情品种选择失败。

## StoreV5

- `dataset_not_found`：manifest 中找不到目标数据集。
- `dataset_has_no_parquet_parts`：数据集存在但没有 parquet part。
- `store_v5_status_failed`：状态检查异常。
- `store_v5_pull_failed`：同步拉取异常。
- `store_v5_pull_exception`：异步拉取任务异常。
- `store_v5_aggregate_failed`：聚合异常。
- `store_v5_query_failed`：查询异常。
- `store_v5_clean_failed`：清理 direct M1 异常。
- `store_v5_delete_failed`：删除 symbol 数据异常。
- `store_v5_aggregated_delete_failed`：删除聚合周期异常。
- `store_v5_m1_gap_repair_failed`：M1 gap 修复异常。

## 前端处理建议

- `bad_request`：提示用户补齐参数或重新选择品种。
- `job_not_found`：提示任务已过期，可重新启动操作。
- `dataset_not_found`：提示先拉取或聚合对应周期。
- MT5 初始化/选择失败：提示检查 MT5 终端登录、品种可见性和交易商后缀。
