## 执行指令：FractalFrame_v5 第一刀：挂载 KLineChart Pro 外壳

### 1. 修改文件

```txt
G:\PythonProject\FractalFrame_v5\frontend\src\App.tsx
```

全部覆盖成：

```tsx
import { useEffect, useRef } from 'react'
import { KLineChartPro, DefaultDatafeed } from '@klinecharts/pro'
import '@klinecharts/pro/dist/klinecharts-pro.css'
import './App.css'

export default function App() {
  const containerRef = useRef<HTMLDivElement | null>(null)

  useEffect(() => {
    if (!containerRef.current) return

    const chart = new KLineChartPro({
      container: containerRef.current,
      symbol: {
        exchange: 'MT5',
        market: 'forex',
        name: 'XAUUSDm',
        shortName: 'XAUUSDm',
        ticker: 'XAUUSDm',
        priceCurrency: 'USD',
        type: 'forex',
      },
      period: {
        multiplier: 5,
        timespan: 'minute',
        text: '5m',
      },
      // 第一刀只验证 Pro 外壳能不能挂起来。
      // 后面会替换成 FractalFrame 自己的 MT5 datafeed。
      datafeed: new DefaultDatafeed(''),
    })

    return () => {
      chart.destroy?.()
    }
  }, [])

  return (
    <div className="ff-v5-root">
      <div ref={containerRef} className="ff-v5-chart" />
    </div>
  )
}
```

## 验收标准

这一刀只看：

```txt
1. 不再显示 Vite 默认 Get started 页面。
2. 页面出现 KLineChart Pro 的图表外壳。
3. 没有 TypeScript 编译报错。
4. 没有白屏。
```

如果报 `DefaultDatafeed` 相关数据错误，可以先不管；因为它默认要 Polygon API key。下一刀我们会写自己的 `createMt5KLineChartProDatafeed.ts`。KLineCharts 的数据接入核心是 `setSymbol / setPeriod / setDataLoader`，其中 `getBars` 负责初始化和分页，最终历史数据需要转成 `KLineData[]`；`timestamp` 必须是毫秒，OHLCV 必须是数字。([KLineChart][2])
